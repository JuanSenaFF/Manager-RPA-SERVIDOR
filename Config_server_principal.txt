# 🚀 Configuração Exclusiva: Servidor 2012 R2

Como o servidor `Windows 2012 R2` não suporta bem o `app.py` (falta de WebView2), nós construímos esta arquitetura de inicialização.

## ✨ Como iniciar o Painel (Sem dar erros de Timeout)
Dentro da pasta `Orquestrador`, eu criei para você o arquivo **`iniciar_orquestrador.bat`**.
Basta dar 2 cliques nele!
1. Ele vai subir a API central (O coração do Orquestrador) na porta **5500**, que é a porta configurada no seu código.
2. Ele vai abrir o Google Chrome automaticamente na tela `http://localhost:5500`.

## ⚙️ Conexão dos Agentes
Para o Agente do servidor 31 (e qualquer outro) conversar com o seu Orquestrador, certifique-se de que o `.env` deles está configurado com o final `:5500` igual a este projeto. Ex:
`MANAGER_URL=http://10.124.100.4:5500/agents/report`
